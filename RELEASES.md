# Релизы

## v1.0.0 — Первый публичный релиз
**Дата:** Апрель 2025

### Функционал:

#### 🧊 Холодильник
Учёт продуктов с категориями (База, Овощи, Фрукты, Молочка, Мясо, Крупы, Заморозка, Хлеб, Сладкое, Мороженое, Еда собаке, Еда кошке). Кнопки +/- для изменения количества.

#### 📅 Календарь
Общий семейный календарь для событий: дни рождения, поездки, напоминания.

#### 📝 Списки
Общие списки покупок. Отмечайте купленное галочкой.

#### 💬 Чаты
- **Семейный чат** — группа для всех членов семьи
- **Личные сообщения** — общение с каждым членом семьи отдельно
- Отправка фото
- Аватары участников

#### 🔐 Авторизация
Доступ только для авторизованных членов семьи. Регистрация через администратора.

#### 📱 PWA
Возможность установить на телефон как приложение.
