# 🏠 Family Hub

**Приватное пространство для вашей семьи.**

Современное веб-приложение для совместного управления семейными делами: холодильник, календарь, списки покупок и семейный чат.

---

## ✨ Возможности

### 📅 Календарь
Общий семейный календарь для событий — дни рождения, поездки, напоминания.

### 🧊 Электронный холодильник
Учёт продуктов с категориями и количеством. Всегда видно что есть дома и что нужно докупить.

### 📝 Списки покупок
Общие списки для всей семьи. Добавляйте, отмечайте купленное.

### 💬 Семейный чат
Telegram-style чат с аватарами участников. Обсуждайте дела не выходя из приложения.

### 🔐 Приватность
Доступ только для авторизованных членов семьи. Данные хранятся в облачной базе Supabase.

---

## 🛠 Технологии

- **Frontend:** React 18 + TypeScript
- **Сборка:** Vite
- **База данных:** Supabase (PostgreSQL)
- **Авторизация:** Supabase Auth
- **Хостинг:** Netlify
- **PWA:** Vite PWA Plugin

---

## 🚀 Запуск локально

```bash
# Клонирование
git clone https://github.com/fonarevw/family-hub.git
cd family-hub

# Установка зависимостей
npm install

# Создание .env файла
cp .env.example .env
# Заполните VITE_SUPABASE_URL и VITE_SUPABASE_ANON_KEY

# Запуск
npm run dev
```

---

## ⚙️ Настройка Supabase

1. Создайте проект на [supabase.com](https://supabase.com)
2. Скопируйте API ключи в `.env`
3. Выполните SQL из `SUPABASE_SETUP.md` для создания таблиц
