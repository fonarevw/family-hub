# Family Hub

Приватное пространство для семьи. Одно приложение вместо десятка чатов, заметок и списков.

---

## Что это

Family Hub — веб-приложение для совместного управления семейными делами. Холодильник, календарь, списки покупок и чат — всё в одном месте.

В отличие от Google Docs, Telegram и заметок — здесь приватность из коробки. Данные видны только авторизованным членам семьи.

---

## Возможности

**Календарь**
Общий календарь для событий: дни рождения, поездки, напоминания.

**Холодильник**
Учёт продуктов. Категории, количество, статус — дома или нужно купить.

**Списки**
Общие списки покупок. Отмечайте купленное, добавляйте новое.

**Чат**
Telegram-style чат с аватарами. Обсуждайте дела не выходя из приложения.

| Фронтенд | TypeScript, React 18, Vite |

---

## Запуск

```bash
git clone https://github.com/fonarevw/family-hub.git
cd family-hub
npm install
cp .env.example .env
```

Заполните `.env`:
```
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

```bash
npm run dev
```

---

## Настройка Supabase

1. Создайте проект на supabase.com
2. SQL Editor → выполните запросы из `SUPABASE_SETUP.md`
3. Скопируйте ключи в `.env`

---

## Установка на телефон

Приложение работает как PWA.

**iPhone:** Safari → Поделиться → На экран Домой

**Android:** Меню браузера → Установить приложение
